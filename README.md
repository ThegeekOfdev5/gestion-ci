# README.md

## Présentation

SaaS de gestion destiné aux PME ivoiriennes et cabinets comptables. Solution conforme OHADA et DGI CI. Positionnement prix accessible. Cible principale. PME locales. Prescripteurs. Experts-comptables.

## Objectifs

• Simplifier la gestion comptable et fiscale
• Automatiser la déclaration TVA
• Réduire les coûts logiciels
• Offrir un support local réactif

## MVP

Modules inclus.
• Authentification
• Gestion entreprise
• Clients et fournisseurs
• Produits
• Facturation
• Comptabilité simplifiée
• Déclaration TVA
• Dashboard
• Abonnements
